# 数据字典

---

## 1. Post 数据字典

### 1.1 说明

表示单条内容数据，可能来自帖子、视频、图文内容。

### 1.2 字段定义

| 字段名 | 中文名 | 类型 | 必填 | 说明 |
|---|---|---|---|---|
| id | 本地记录ID | string | 是 | 本地数据库主键 |
| platform | 平台 | string | 是 | 如 xhs / dy / ks |
| postId | 帖子ID | string | 是 | 平台侧内容唯一标识 |
| postType | 内容类型 | string | 否 | video / image / mixed / text |
| title | 标题 | string | 否 | 页面可见标题 |
| content | 内容描述 | string | 否 | 正文/文案 |
| url | 帖子链接 | string | 是 | 原始页面链接 |
| shortUrl | 短链接 | string | 否 | 如平台提供 |
| coverUrl | 封面图 | string | 否 | 主封面图 URL |
| publishTime | 发布时间 | string | 否 | 原始发布时间 |
| authorId | 作者ID | string | 否 | 对应作者唯一标识 |
| authorName | 作者昵称 | string | 否 | 作者展示名称 |
| authorUrl | 作者主页链接 | string | 否 | 作者主页地址 |
| likeCount | 点赞数 | number | 否 | 页面可见值 |
| commentCount | 评论数 | number | 否 | 页面可见值 |
| collectCount | 收藏数 | number | 否 | 页面可见值 |
| shareCount | 分享数 | number | 否 | 页面可见值 |
| viewCount | 播放/浏览数 | number | 否 | 部分平台有 |
| mediaCount | 媒体数量 | number | 否 | 图片或视频资源数量 |
| tags | 标签 | string[] | 否 | 内容标签 |
| sourcePageUrl | 来源页面URL | string | 是 | 触发采集时页面地址 |
| sourcePageType | 来源页面类型 | string | 否 | post_detail / profile / feed |
| collectedAt | 采集时间 | string | 是 | 本地采集时间 |
| updatedAt | 更新时间 | string | 是 | 最近更新时间 |
| isFavorite | 是否标星 | boolean | 否 | 本地标记 |
| note | 备注 | string | 否 | 本地备注 |
| extra | 扩展字段 | object | 否 | 平台差异字段 |

### 1.3 唯一性规则

建议唯一键：
`platform + postId`

## 4. Media 数据字典

### 4.1 说明

表示帖子下关联的媒体资源。

### 4.2 字段定义

| 字段名 | 中文名 | 类型 | 必填 | 说明 |
|---|---|---|---|---|
| id | 本地记录ID | string | 是 | 本地主键 |
| platform | 平台 | string | 是 | 所属平台 |
| postId | 所属帖子ID | string | 是 | 关联帖子 |
| mediaId | 媒体ID | string | 否 | 平台或本地生成 |
| mediaType | 媒体类型 | string | 是 | image / video |
| url | 媒体链接 | string | 是 | 原始资源链接 |
| coverUrl | 封面链接 | string | 否 | 视频封面或图片缩略图 |
| fileName | 文件名 | string | 否 | 下载生成文件名 |
| ext | 文件扩展名 | string | 否 | jpg/png/mp4 |
| width | 宽 | number | 否 | 分辨率宽 |
| height | 高 | number | 否 | 分辨率高 |
| duration | 时长 | number | 否 | 视频时长（秒） |
| downloadStatus | 下载状态 | string | 否 | pending / success / failed |
| sourcePageUrl | 来源页面URL | string | 是 | 采集来源页 |
| collectedAt | 采集时间 | string | 是 | 本地采集时间 |
| updatedAt | 更新时间 | string | 是 | 本地更新时间 |
| extra | 扩展字段 | object | 否 | 平台差异字段 |

### 4.3 唯一性规则

建议唯一键：
优先 `platform + mediaId`，没有 mediaId 时用 `platform + postId + url`

## 5. Task 数据字典

### 5.1 说明

统一记录采集、导出、下载行为。

### 5.2 字段定义

| 字段名 | 中文名 | 类型 | 必填 | 说明 |
|---|---|---|---|---|
| id | 任务ID | string | 是 | 本地任务唯一 ID |
| taskType | 任务类型 | string | 是 | collect_post / collect_author / collect_comments / export_data / download_media |
| platform | 平台 | string | 否 | 对应平台 |
| targetId | 目标ID | string | 否 | 帖子 ID / 作者 ID 等 |
| targetUrl | 目标URL | string | 否 | 任务来源地址 |
| title | 任务标题 | string | 否 | 用于 UI 展示 |
| status | 状态 | string | 是 | pending / running / success / failed / canceled |
| progress | 进度 | number | 否 | 0-100 |
| totalCount | 总数 | number | 否 | 批量任务总条数 |
| successCount | 成功数 | number | 否 | 已成功数 |
| failedCount | 失败数 | number | 否 | 已失败数 |
| errorMessage | 错误信息 | string | 否 | 最近错误 |
| createdAt | 创建时间 | string | 是 | 任务创建时间 |
| updatedAt | 更新时间 | string | 是 | 任务更新时间 |
| finishedAt | 完成时间 | string | 否 | 任务结束时间 |
| meta | 附加信息 | object | 否 | 任务参数、导出配置等 |
