### 产品决策

1. MVP 不包含 AI、云同步、团队协作
2. 优先实现小红书平台
3. 导出优先级：Excel > CSV > JSON
4. 本地存储优先，不做后端

### 技术决策

1. 使用 WXT 框架开发 Chrome 插件
2. 使用 Zustand 做状态管理
3. 使用 IndexedDB 做本地存储
4. 使用 SheetJS 生成导出文件

### 开发决策

1. P0 完成单平台采集+导出
2. P1 完成作者采集+任务中心
3. P2 完成多平台扩展
